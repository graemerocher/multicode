---
name: java-install
description: Install locations for Java and GraalVM versions. Load when you require a specific Java version.
---

Use an existing Java version from any of the following locations:

- `/usr/lib/jvm`
- `/usr/lib64/graalvm`
- `~/.sdkman/candidates/java`

If an appropriate Java version is not available, you must prompt the user. One possibility is fetching the version from 
[temurin releases](https://adoptium.net/temurin/releases).
